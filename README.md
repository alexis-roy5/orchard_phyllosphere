# orchard_phyllosphere
**Investigateurs principaux:** Sophie Boutin, Alexis Roy, Jonathan Rondeau-Leclaire, Isabelle Laforest-Lapointe. 

**Prise de données:** Sur plusieurs verges, Sophie a pris des échantillons de feuilles et de fleurs à différents timepoints dans le but de caractériser le mycobiome. 
	-> Année de prise de données: 2023
	
**Objectif du projet:** Caractériser le mycobiome des vergers de pommes pour mieux comprendre.

**Contenu du repository:** On y trouve les données brutes et les tables de taxonomy/séquences sur IP34. L'objet phyloseq raréfié. Dès l'entrée, on trouve un dossier **2023**, **out** et **scripts**. 
	**2023** pour DADA2.
		-> Structure pour pipeline DADA2: 2023/data/barcode/<directories_for_DADA2>
		-> Out de DADA2: 2023/out/.
		**final** pour les figures du papier.
		**scripts** pour les scripts utilisés pour ce projet.
