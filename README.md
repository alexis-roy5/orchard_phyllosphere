# orchard_phyllosphere 2023
**Investigators** Sophie Boutin, Alexis Roy, Jonathan Rondeau-Leclaire, Isabelle Laforest-Lapointe. 

## Datasets
Samples taken on the leaves and flowers of multiple apple trees in the Eastern townships across the 2023 growing season. Raw data available at https://www.ebi.ac.uk/ena/browser/view/PRJEB96463

## Methods
Amplicon sequencing of ITS gene and ASV processing using DADA2.

Manuscript in Revision. Current version available at https://www.biorxiv.org/content/10.1101/2025.09.25.678622v2
